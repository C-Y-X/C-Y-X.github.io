部分神舟及蓝天模具笔记本指纹模块不识别bug

发表于 2019-07-28 更新于 2020-07-28 阅读次数： Valine：
本文字数： 170 阅读时长 ≈ 1 分钟

故障表现

部分神舟笔记本及蓝天模具笔记本指纹模块经常在开机一段时间后不能识别指纹，重启电脑或者禁止指纹模块再重新启用后会恢复正常。

引起原因及解决方法

应该是windows在待机状态下 启动节能模式 关闭非必要设备 存在bug指纹模块不能正常被唤醒。
Windows下 打开设备管理器 选择生物识别设备 右键属性 电源管理 将允许计算机关闭此设备以节约电源 中√ 去掉
